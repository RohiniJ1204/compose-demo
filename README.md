# Docker Compose Application - AWS EC2 Deployment
## Project overview 
This project is a **Docker compose-based multi container application** consisting of a backend and frontend.
The application was developed and tested locally and later **deployed on an AWS EC2 instance** to provide public access.

-----

## Technologies Used
- Docker
- Docker Compose
- AWS EC2
- Linux (Ubuntu)
- Backend: Flask
- Frontend: HTML
